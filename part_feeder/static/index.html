Orienting Polygonal Parts without Sensors: An Implementation in Python


This website is an implementation of Prof. Ken Goldberg's 1993 PhD thesis, Orienting Polygonal Parts Without Sensors. To begin, click "New Part" to clear the canvas and begin clicking points on the canvas to draw a polygon. Once you have completed drawing your polygon, click the first point on the polygon to complete it and press the "Next" button below the canvas to see the algorithm in action. Please note that this website does not work on mobile platforms.
